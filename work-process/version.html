# 版本规范

<主版本号>.<次版本号>.<修订版本号>

  • 主版本号：当功能模块有较大的变动，比如增加多个模块或者整体架构发生变化
  • 次版本号：当功能有一定的增加或变化，比如增加了对权限控制、增加自定义视图等功能
  • 修订版本号: Bug 修复或是一些小的变动，要经常发布修订版，时间间隔不限，修复一个严重的bug即可发布一个修订版。
